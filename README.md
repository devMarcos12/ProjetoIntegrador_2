## 🚀 Instalando dependências

Para isntalar as dependências, siga estas etapas:

Linux e macOS:

```
npm install
```

Windows:

```
npm install
```

## ☕ Usando AcademiaX

Para usar Academiax, siga estas etapas:

Acesse o diretório (oracle-node-sample):
```
cd oracle-node-sample
```
Depois acesse os endpoints:

```
npm run dev
```
